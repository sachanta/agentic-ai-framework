  How It Helps The Project                                                                                                                                                                                                     
                                                                                                                                                                                                                               
  Phase 3 is the content discovery engine that powers the entire newsletter generation pipeline. Without it, there's no source material for newsletters.                                                                       
                                                                                                                                                                                                                               
  The flow:                                                                                                                                                                                                                    
  1. User provides topics (e.g., "AI", "climate tech", "startup funding")                                                                                                                                                      
  2. Tavily service searches the web for recent, high-quality articles on each topic                                                                                                                                           
  3. Results are filtered (remove duplicates, low-quality sources, old content)                                                                                                                                                
  4. AI generates summaries of each article                                                                                                                                                                                    
  5. This curated content feeds into the Research Agent (Phase 6)                                                                                                                                                              
  6. The Research Agent presents articles at HITL Checkpoint 1 for user approval                                                                                                                                               
  7. Approved articles go to the Writing Agent (Phase 7) to generate the newsletter                                                                                                                                            
                                                                                                                                                                                                                               
  Why Tavily? It's an AI-native search API designed for LLM applications - returns clean, structured results with built-in content extraction, unlike raw web scraping. The advanced search depth provides deeper content      
  analysis and better relevance scoring.
