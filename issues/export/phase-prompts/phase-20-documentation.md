# Phase 20: Documentation & Deployment

## Goal
Complete documentation and add deployment configuration.

---

## Copilot Prompt

```
You are helping finalize documentation and deployment config for an application.

### Context
- README should help new developers get started quickly
- Docker configuration for easy deployment
- API documentation is auto-generated by FastAPI

### Files to Read First
- Existing README.md (if any)
- backend/app/main.py (FastAPI app)
- frontend/package.json (scripts)

### Implementation Tasks

1. **Create/Update `README.md` in project root:**
   ```markdown
   # Agentic AI Framework

   A full-stack framework for building multi-agent AI applications.

   ## Quick Start

   ### Prerequisites
   - Python 3.11+
   - Node.js 18+
   - MongoDB
   - Ollama (optional, for LLM features)

   ### Backend Setup
   ```bash
   cd backend
   python -m venv .venv
   source .venv/bin/activate  # or .venv\Scripts\activate on Windows
   pip install -e ".[dev]"
   uvicorn app.main:app --reload
   ```

   ### Frontend Setup
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

   ### Environment Variables
   Create `backend/.env`:
   ```env
   MONGODB_URI=mongodb://localhost:27017
   MONGODB_DATABASE=agentic_ai
   LLM_PROVIDER=ollama
   LLM_DEFAULT_MODEL=llama3
   ```

   ## API Documentation
   - Swagger UI: http://localhost:8000/api/v1/docs
   - ReDoc: http://localhost:8000/api/v1/redoc

   ## Architecture
   See `docs/architecture.md` for detailed architecture documentation.

   ## Testing
   ```bash
   # Backend tests
   cd backend && .venv/bin/python -m pytest -v

   # Frontend (manual for now)
   cd frontend && npm run dev
   ```
   ```

2. **Create `backend/Dockerfile`:**
   ```dockerfile
   FROM python:3.11-slim

   WORKDIR /app

   COPY pyproject.toml .
   RUN pip install --no-cache-dir .

   COPY app/ app/

   EXPOSE 8000

   CMD ["uvicorn", "app.main:app", "--host", "0.0.0.0", "--port", "8000"]
   ```

3. **Create `frontend/Dockerfile`:**
   ```dockerfile
   FROM node:18-alpine as build

   WORKDIR /app
   COPY package*.json ./
   RUN npm ci
   COPY . .
   RUN npm run build

   FROM nginx:alpine
   COPY --from=build /app/dist /usr/share/nginx/html
   EXPOSE 80
   CMD ["nginx", "-g", "daemon off;"]
   ```

4. **Create `docker-compose.yml` in project root:**
   ```yaml
   version: '3.8'

   services:
     backend:
       build: ./backend
       ports:
         - "8000:8000"
       environment:
         - MONGODB_URI=mongodb://mongo:27017
         - MONGODB_DATABASE=agentic_ai
         - LLM_PROVIDER=ollama
         - OLLAMA_BASE_URL=http://host.docker.internal:11434
       depends_on:
         - mongo

     frontend:
       build: ./frontend
       ports:
         - "80:80"
       depends_on:
         - backend

     mongo:
       image: mongo:7
       ports:
         - "27017:27017"
       volumes:
         - mongo_data:/data/db

   volumes:
     mongo_data:
   ```
