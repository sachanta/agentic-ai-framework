Phase 5: Data Management APIs - Complete                                                                                                                                                                                     
                                                                                                                                                                                                                               
  MongoDB Endpoints (/api/v1/mongodb)                                                                                                                                                                                          
                                                                                                                                                                                                                               
  - Status & Stats: /status, /stats                                                                                                                                                                                            
  - Collection CRUD: /collections (list, create, delete)                                                                                                                                                                       
  - Index Management: /collections/{name}/indexes                                                                                                                                                                              
  - Document CRUD: Full pagination support with has_next/has_prev                                                                                                                                                              
  - Query & Aggregation: Advanced filtering, projection, sorting                                                                                                                                                               
  - Bulk Operations: Import/export                                                                                                                                                                                             
  - Agentic AI Features:                                                                                                                                                                                                       
    - Agent Executions (/agent-executions) - track agent runs                                                                                                                                                                  
    - Conversations (/conversations) - conversation memory                                                                                                                                                                     
    - Knowledge Base (/knowledge-base) - document storage for RAG                                                                                                                                                              
    - Agent Configs (/agent-configs) - agent configurations                                                                                                                                                                    
    - Metrics (/metrics) - analytics with aggregation                                                                                                                                                                          
                                                                                                                                                                                                                               
  Weaviate Endpoints (/api/v1/weaviate)                                                                                                                                                                                        
                                                                                                                                                                                                                               
  - Status & Stats: /status, /stats                                                                                                                                                                                            
  - Collection CRUD: Create, list, get, delete collections                                                                                                                                                                     
  - Document CRUD: Full CRUD with pagination and vectors                                                                                                                                                                       
  - Search Types:                                                                                                                                                                                                              
    - /search/text - text-based near text search                                                                                                                                                                               
    - /search/vector - direct vector search                                                                                                                                                                                    
    - /search/embedding - generates embeddings from text (uses Ollama)                                                                                                                                                         
    - /search/hybrid - keyword + vector combined (alpha parameter)                                                                                                                                                             
  - Bulk Operations: Import with/without vectors, generate embeddings                                                                                                                                                          
  - RAG Support (/rag/*):                                                                                                                                                                                                      
    - Add documents with auto-embedding generation                                                                                                                                                                             
    - Query with semantic search                                                                                                                                                                                               
    - List and delete documents                                                                                                                                                                                                
  - Agent Memory (/memory/*):                                                                                                                                                                                                  
    - Store agent memory entries with vectors                                                                                                                                                                                  
    - Query by semantic similarity                                                                                                                                                                                             
    - Filter by agent/session/memory type                                                                                                                                                                                      
    - Clear agent memories                                                                                                                                                                                                     
                                                                                                                                                                                                                               
  Key Features for Frontend Integration                                                                                                                                                                                        
                                                                                                                                                                                                                               
  - Consistent pagination format: {items, total, page, page_size, total_pages, has_next, has_prev}                                                                                                                             
  - Proper HTTP status codes and error handling                                                                                                                                                                                
  - All responses are JSON-serializable                                                                                                                                                                                        
  - Support for filtering, sorting, and projection                                                                                                                                                                             
  - Auto-generation of embeddings using the configured embeddings provider
