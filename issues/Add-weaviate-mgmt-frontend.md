 # Key classes:                                                                                                                                                                                                                
 class TextChunk:                                                                                                                                                                                                              
     content: str

     page_number: int                                                                                                                                                                                                          
                                                                                                                                                                                                                               
 class PDFProcessor:                                                                                                                                                                                                           
     async def extract_text(self, content: bytes) -> Tuple[str, int]  # text, page_count                                                                                                                                       
     async def chunk_text(self, text: str, chunk_size: int, overlap: int) -> List[TextChunk]                                                                                                                                   
                                                                                                                                                                                                                               
 - Use pypdf library (pure Python, no external deps)                                                                                                                                                                           
 - Chunk size: 1000 chars default, 200 char overlap                                                                                                                                                                            
 - Respect sentence boundaries when possible                                                                                                                                                                                   
                                                                                                                                                                                                                               
 app/services/upload_manager.py                                                                                                                                                                                                
                                                                                                                                                                                                                               
 Background processing and progress tracking.                                                                                                                                                                                  
 class UploadManager:                                                                                                                                                                                                          
     async def start_upload(upload_id, content, config) -> None                                                                                                                                                                
     async def get_progress(upload_id) -> AsyncGenerator[Progress]                                                                                                                                                             
     async def cancel_upload(upload_id) -> None                                                                                                                                                                                
                                                                                                                                                                                                                               
 - Uses asyncio.Queue for progress events

 Processing Flow                                                                                                                                                                                                               
                                                                                                                                                                                                                               
 1. User drops PDF file                                                                                                                                                                                                        
 2. Frontend validates: .pdf extension, <50MB                                                                                                                                                                                  
 3. POST /rag/upload-pdf with FormData                                                                                                                                                                                         
 4. Backend returns upload_id immediately                                                                                                                                                                                      
 5. Frontend connects to SSE: GET /progress/{upload_id}                                                                                                                                                                        
 6. Backend processes:                                                                                                                                                                                                         
    a. Extract text with pypdf (progress: 0-20%)                                                                                                                                                                               
    b. Chunk text (progress: 20-30%)                                                                                                                                                                                           
    c. Generate embeddings in batches (progress: 30-90%)                                                                                                                                                                       
    d. Store to Weaviate RAG collection (progress: 90-100%)                                                                                                                                                                    
 7. Each stage emits progress event via SSE                                                                                                                                                                                    
 8. Frontend updates progress bar and stage label                                                                                                                                                                              
 9. On completion/failure, SSE connection closes                                                                                                                                                                               
 10. Frontend shows result with chunk count
